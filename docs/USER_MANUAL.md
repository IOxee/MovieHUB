# User Manual - MediaHub

This document serves as a visual guide for using the MediaHub application, highlighting its mobile responsiveness and intelligent recommendation system.

## 1. Main View (Desktop)
The home screen offers quick access to the entire movie and series catalog.

**Suggested screenshot:** Full desktop web view showing the timeline, filters, and cover grid.
![Desktop View](images/desktop_home.png)

### Features:
- **Content Filters:** Toggle between "All", "Movies", and "Series".
- **Timeline:** Double slider to filter content by year (1970 - 2026).
- **Instant Search:** Real-time search by title.

---

## 2. Mobile Adaptation
The application is fully optimized for mobile devices, with a friendly touch interface.

**Suggested screenshot:** Mobile inspection view (iPhone/Android).
![Mobile View](images/mobile_home.png)

### Mobile Improvements:
- **Compact Header:** Elements reorganized for easy thumb reach.
- **Touch User Menu:** Profile button that responds to tap instead of hover.
- **Adjusted Sliders:** Timeline controls with larger contact area.

---

## 3. Recommendation System (AI)
The heart of MediaHub. Pressing the "AI" (or magic wand) button transforms the interface to show suggested content based on your tastes.

**Suggested screenshot:** Screen with recommendation interface active (without search bar or timeline).
![Recommendations Mode](images/recommendations_mode.png)

### AI Mode Features:
- **Clean Interface:** Search bar and timeline are hidden to avoid distractions.
- **Contextual Navigation:**
  - **"Again" Button**: Instantly generates new recommendations.
  - **"Return to Catalog" Button**: Returns to normal view preserving your filters.
- **Smart Filters:** If you switch from "Movies" to "TV" while in this mode, the AI automatically recalculates recommendations for that media.

---

## 4. Details and Voting
Clicking on any cover opens a detailed card.

**Suggested screenshot:** Open movie modal.
![Movie Detail](images/movie_detail.png)

### Voting System:
- **Registered Users:** Can vote with "Hate", "Dislike", "Like", or "SuperLike". These ratings feed the AI algorithm.
- **Guest Users:** See a message inviting them to log in, without showing irrelevant affinity percentages.
![Guest Notice](images/guest_voting.png)
